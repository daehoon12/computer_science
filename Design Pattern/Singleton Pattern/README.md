# Singleton Pattern

## Singleton Pattern

### Problems  
- 여러 객체가 생성되면 상태 관리가 어려움.  

### Soultions
- 객체 생성사를 중앙 관리  

### Goals  
- 클래스가 한 개의 인스턴스만을 가질 수 있게 하고, 어디서나 생성된 인스턴스에 접근할 수 있도록 함.  
- 객체가 1개라서 일관된 상태.  

### 고전적 구현 방법  
- Private 디폴트 생성자 구현  
- 싱글턴 인스턴스를 저장하는 정적 변수 생성  
- 싱글턴 인스턴스를 반환하는 정적 팩토리 메소드 구현
