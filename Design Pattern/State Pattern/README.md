# State Pattern

## State Pattern

### 문제 
- State가 여러 개 있고, if문으로 State를 통제  

### 해결 방안  
- 상태를 한 곳에서 관리  

### 결과  
- 변경 최소화

### State (상태)  
- 시점에 따라 특정 상탱 있어야 함. 처음에 가지게 되는 초기 상태 (State) 또는 상황에 따라 여러 상태 가운데 한 상태를 가질 수 있음.  
### Transition (전이)  
- 외부 입력에 따라 가능한 상태로 전환.
