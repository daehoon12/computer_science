# Artificial Intelligence  

## 용어 정리  

### 학습 (Learning)
- 데이터의 규칙을 컴퓨터 스스로 찾아내는 것.  

### 모델 (Model)
- 학습을 통해 만들어진 프로그램  

### 훈련 데이터 (Training Data)  
- 모델을 훈련시키기 위한 데이터  
- '입력'과 '타겟'으로 구성  

#### 입력  
- 모델이 풀어야 할 일종의 문제 같은 것  
#### 타겟  
- 모델이 맞춰야 할 정답
